# Quiz App

A small front-end quiz app built with plain HTML, CSS, and JavaScript.

## What It Does

- Shows 5 simple math questions
- Displays one question at a time
- Tracks the score as a percentage
- Randomizes the question order each time the page loads

## Files

- `index.html` contains the page structure
- `style.css` contains the styling
- `app.js` contains the quiz logic and questions

## How To Run

This project does not need any build tools or dependencies.

1. Clone or download the project
2. Open `index.html` in your browser

You can also run it with a simple local server if you prefer.

## Notes

- The quiz currently uses hardcoded questions in `app.js`
- The score is shown as a percentage after all questions are answered
